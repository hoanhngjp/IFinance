8            Package fastapi-pagination:                                                                                                                                                           Thư viện hỗ trợ phân trang chuẩn, giúp API Transaction trả về { items: [...], total: 100, page: 1, size: 50 } tự động.

11                  Logic Refresh Token:                                                                                   - Access Token: Sống ngắn (ví dụ: 15 phút - 1 tiếng).\n- Refresh Token: Sống dài (ví dụ: 7 ngày - 30 ngày). Lưu trong DB (hoặc Redis) để có thể thu hồi (revoke) nếu User logout hoặc bị hack.
12         Logic Transfer (Chuyển tiền):                                                      Backend phải đảm bảo Atomic Transaction: Nghĩa là việc (Trừ tiền Ví A) và (Cộng tiền Ví B) phải cùng thành công hoặc cùng thất bại. Không được phép xảy ra chuyện Ví A trừ rồi mà Ví B chưa nhận được (lỗi tiền "bốc hơi").
13   Logic Bán Đầu Tư (Sell Investment):                                                                          - Khi bán, trạng thái Investment chuyển sang closed.\n- Hệ thống tự tính: Profit = Selling Price - Principal Amount.\n- Tự tạo một Transaction loại investment_return để ghi nhận dòng tiền quay về ví.
